# ספר המתכונים של קירה

אתר סטטי קטן עם אוסף המתכונים, מוכן לאחסון ב-GitHub Pages.

## מבנה הקבצים

- `index.html` – העיצוב ותצוגת המתכונים. בדרך כלל לא נוגעים בו.
- `recipes.json` – **כל המתכונים נמצאים כאן**. זה הקובץ היחיד שעורכים כשמוסיפים מתכון חדש.
- `images/` – תמונות מצורפות (אם רוצים שתמונה תופיע בכרטיסיית מתכון, מעלים אותה לתיקייה הזו ומפנים אליה ב-`recipes.json`).

## איך מוסיפים מתכון חדש

עורכים את `recipes.json` ב-GitHub (לחיצה על קובץ → אייקון העיפרון), מוסיפים אובייקט חדש למערך, שומרים (commit). תוך כדקה האתר מתעדכן אוטומטית.

מבנה אובייקט מתכון בודק:

```json
{
  "id": 100,
  "title": "שם המתכון",
  "date": "5/20/26",
  "cat": "desserts",
  "cats": ["desserts"],
  "src": { "type": "site", "label": "10 דקות", "icon": "🍳" },
  "urls": ["https://example.com/recipe"],
  "imgs": [],
  "html": "<p>תוכן המתכון בפורמט HTML</p>",
  "s": "שם המתכון תוכן לחיפוש"
}
```

קטגוריות אפשריות (השדה `cat`): `breads`, `salads`, `soups`, `rice`, `chicken`, `meat`, `meatballs`, `fish`, `pasta`, `eggs_dairy`, `sauces`, `desserts`, `drinks`, `other`.
